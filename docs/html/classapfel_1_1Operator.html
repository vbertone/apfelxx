Detailed Description

This class defines the basic object Operator which essentially contains the convolution bewteen an "Expression" object (e.g. a splitting function) and a set of interpolanting functions defined on a grid.

Note
Both the forward and the non-forward cases can be handled by this class. The correspond respectively to inclusive (DGLAP-like) and exclusive (GPD-like) processes.
Examples
distribution_test.cc, massiveCF_test.cc, operator_test.cc, principal_value_test.cc, and set_test.cc.
