Detailed Description

Collection of the charged-current massive coefficient functions for F2, F2, and xF3 to O(αs). Expressions taken from https://arxiv.org/pdf/hep-ph/9603304.pdf.

Note
In the following 'xi' indicates the ratio Q2 / M2.
