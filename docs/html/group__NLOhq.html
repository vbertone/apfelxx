Detailed Description

Collection of the massive coefficient functions for processes with a massive heavy quark in the intial state. These coefficient functions can be used both in the neutral-current and in the charged-current cases.

Note
All the expressions of the one-loop coefficient functions are extracted from https://arxiv.org/pdf/hep-ph/9805233.pdf. The coefficient function C is witten in terms of the private _R function such that:

C(x) = ( R(x) - _R(1) ) / ( 1 - x ) + _R(x) / ( 1 - x )+ + ( L + _R(1) ) delta( 1- x )
