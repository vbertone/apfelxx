Detailed Description

Collection of the Zero-mass coefficient functions for F2, FL, and F3 up to O(αs2).

Note
While for the O(αs) coefficient functions exact expressions are used, a fast parameterisation for the O(αs2) ones is used. See https://www.liverpool.ac.uk/~avogt/coeff.html for more details.
