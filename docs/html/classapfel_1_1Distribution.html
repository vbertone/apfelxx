Detailed Description

The Distribution class defines one of the basic objects of APFEL++. This is essentially the discretisation of a function that can be conveniently used for convolutions.

Examples
distribution_test.cc, doubleobjint_test.cc, interpolation_test.cc, principal_value_test.cc, qgriddist_test.cc, structurefunction_massive_test.cc, and structurefunction_test.cc.
