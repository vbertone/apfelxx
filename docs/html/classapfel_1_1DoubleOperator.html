Detailed Description

This class defines the basic object DoubleOperator which essentially contains the convolution bewteen a "DoubleExpression" object and two sets of interpolanting functions defined on two independent grids. This is the two-dimensional generalisation of the object "Operator".

Note
For the moment, only the inclusive massless case can be handled within this class.
Examples
sidis_test.cc.
